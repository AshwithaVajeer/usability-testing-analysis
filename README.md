# 📝 To-Do List Mobile App Prototype

## 📌 Project Overview
This project is a simple and interactive To-Do List mobile app prototype designed using Figma. The application helps users manage daily tasks through a clean and user-friendly interface.

# 🎯 Objective
The objective of this project is to design and test a mobile UI prototype while improving usability and user experience through feedback analysis.

# 🛠 Tool Used
- Figma

# ✨ Features
- Interactive mobile prototype
- Simple navigation flow
- Add task screen
- Profile screen
- Smooth animations
- User-friendly UI

# 📱 Screens Included
- Splash Screen
- Login Screen
- Home Screen
- Add Task Screen
- Profile Screen

# 🧪 Usability Testing
Usability testing was conducted with 5 users to analyze navigation flow, button visibility, and interface clarity.

# 📊 Feedback Summary
| Observation | Result |
| Navigation easy to use | Positive |
| Add button visibility | Needs Improvement |
| Clean interface | Positive |

# 🔧 Improvements Made
- Increased button size
- Improved spacing
- Enhanced navigation icons
- Added better visual hierarchy
